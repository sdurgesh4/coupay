How to Add Coupon

  • 1. Copy coupon details from GPay or voucher app.
  • 2. Enter brand name and discount description.
  • 3. Paste coupon code.
  • 4. Add expiry date.
  • 5. Paste redeem link in Redeem URL if available.
  • 6. Upload so others can use it before expiry.
